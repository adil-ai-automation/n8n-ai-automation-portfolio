# AI Gmail Draft Assistant

## Overview

The AI Gmail Draft Assistant is an n8n workflow that uses OpenAI to generate professional email drafts and automatically save them as Gmail drafts. It helps users compose clear, well-structured emails without manually writing each message.

---

## Problem

Writing professional emails repeatedly can be time-consuming and inconsistent. Employees often spend valuable time drafting routine emails instead of focusing on higher-value work.

---

## Solution

This workflow acts as an AI email assistant by understanding the user's request, generating a polished email, and saving it directly to Gmail as a draft.

The generated draft includes:

- Professional subject line
- Personalised greeting
- Well-structured email body
- Professional closing
- Grammar and spelling correction

---

## Business Value

This workflow helps businesses:

- Reduce time spent writing emails
- Improve communication quality
- Maintain consistent professional writing
- Increase employee productivity
- Standardise business correspondence

---

## Technology Stack

- n8n
- OpenAI GPT-5
- AI Agent
- Gmail API
- Prompt Engineering
